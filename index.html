Snake

While learning javascript, I made a two dimensional Snake game, much like the arcade version. This game runs in your browser and is built using p5.js, a javascript library. This game also keeps keeps track of your current score as well as your highest score while you play. 2 Player Pong

2 Player Pong is another personal project where, with the help of some tutorials, I used javascript and p5.js in order to build a visual game. This game is very similar to the arcade video game Pong, where two players use their paddles to hit a ball back and forth, in order to try and score points. This game also keeps tracks of the player scores, and it also displays a message showing who won once a player reaches 5 points.

PHP Math Game

This is a project which I worked on for a school assignment. This is a simple Math Game that was made entirely with PHP. The game picks two random numbers between 0 and 50, and tells the user to either add them or subtract them. It then informs the reader if their answer was correct or not. Another feature of this game is that it contains a functioning login page, and without logging in with the correct credentials, it is impossible to access the game.

Random Quote Generator

This is a personal project that I made to further my understanding of AJAX, JSON and APIs. This project uses the Forismatic API and AJAX requests to get random quotes and display them to users. no quotes have been hardcoded, and all quotes are fetched from the API randomly. Furthermore, This page changes to a random color each time the "get new quote button" is pressed. I also added a button that lets the user tweet the current quote that they are viewing.
The idea for this project came from FreeCodeCamp.
